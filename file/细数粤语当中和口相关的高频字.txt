代词：方位代词






呢

呢 = 这

呢度系边到啊？ = 这里是哪里啊？






 = 那

窀鱿地克啊？ = 那个人是什么来头啊？/那个人是谁？






时态








 = 完、过，表示完成。

我食返啦！ = 我吃完饭/我吃过饭。






语气







 = 的（用在陈述句和问句，可在句中也可在句末。使用频率最高）

呢野唔好食噶。 = 这样的东西不好吃。

好 = 好的。




噶

噶 = 的（用在问句和陈述句，在句末）

咩你罡粒 = 为什么你这样的？

好噶 = 好的。（这个使用频率较少，没“好”使用频率高）




喔 啵

喔、啵 = 啊（用在问句。放在句末。别人和你说完话之后，你带有轻微的怀疑而说出的问句，感情色彩和语气都不是很强。）

盍肃浮盍肃 = 这样啊






 = 吧（用在问句。放在句末。别人和你说完话之后，你带有震惊而说出的问句。这个语气比喔、啵要强）

唔系 = 不是吧？






 = 啊（用在问句和一般陈述句。放在句末。这个字一般用在问句比较多，一般是撒娇说话时用的。当用在陈述句的时候，语句带有一点点看不起的蔑视感觉）

买窦衫比我啦，好唔好ǎ = 买好不好啊？

件衫质量好差ā = 那件衣服的质量很差嘛。




哇

用在陈述句。单字使用。表示惊讶时就用这个字。感情色彩最重。




咦

用在问句。单字使用。表示惊讶时就用这个字。感情色彩比较重。用这个咦字还可以唱歌哦，谁知道的？




啊

用在问句。单字使用。这个感情色彩比较轻，主要用于别人叫你，就用啊来回答，等于国语的什么事。




q

用在陈述句。单字使用。表示非常蔑视，感情色彩重。






形容词、副词






咩

咩 = 什么（使用频率极高）

咩啊？ 咩野？ = 什么？

做咩啊？做咩S？ = 干什么？





 = 这么、这样（使用频率极高）

疃嫉冒。 = 这样也行啊？




唔

唔 = 不（使用频率极高）

唔系 = 不是。

唔该晒 = 谢谢。这个使用频率极度非常高，是别人对你帮助之后，你表示感谢的。

唔知 = 不清楚。 唔知可以结合哪个语气字组成陈述句？






 = 是（读第四声）、在（读第一声）。使用频率极高。

唔 = 不是。

我Simon。 = 我是simon。

者眨 = 是不是？

你毡甙。 = 你在哪里啊？



S

S = 东西。

你好S。 = 你好啊你（带有调侃味道的蔑视感觉）

晚黑食咩S？ = 晚上吃什么东西？






 = 对、正确、巧合。

你系岩。 = 你是对的。

咦，？ = 咦，那么巧？

我先食饭。（不懂这句是什么意思的加Simon私人微信问）







动词








 = 来。

咩噶？ = 什么来的？

快入个碗到度。 = 快进碗里来。





喊

喊 = 哭。

你喊啦，喊出会舒服噶。 = 你哭吧，哭出来会舒服点的。




嗌

嗌 = 叫。

你嗌啦，你嗌破喉咙都尤死砟愀晾玻∑坪砹，破喉咙 = 你叫吧，你叫破喉咙都没人理你的！
